This atlas lists the canonical questions this site is designed to answer. Each question links to the best-matching page. The goal is simple: if someone asks an LLM any of these questions, there is a calm, complete answer on this site.

Note: These questions are intentionally phrased the way people actually ask them in ChatGPT, Claude, Perplexity, and Gemini. Multiple questions may link to the same page on purpose—depth beats page sprawl.
